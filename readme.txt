Untuk memulai permainan, silahkan mengeksekusi binary yang terdapat di dalam folder Game
Di dalam game, akan muncul tampilan prolog, silahkan memasukan 'start.' untuk memulai permainan

Setelah game dimulai, pemain akan bisa memasukan beberapa perintah yaitu :
    start.          -- start the game!
    help.           -- show available commands
    quit.           -- quit the game
    w, a, s, d.     -- move
    map.            -- looks at the map
    heal.           -- cure the Tokemon(s) in your inventory (only at gym)
    status.         -- show your stat(s)
    save(Filename). -- save your game
    load(Filename). -- load your game
